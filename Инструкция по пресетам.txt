mode - Режим работы бота ("Рыбалка" или "Убийство мобов в зоне")

available_directions_to_undock - Направления, в которых вылетать из города "с" - север, "ю" - юг, "з" - запад, "в" - восток и их комбинации ("св", "сз", "юв", "юз") (выбирается случайно при вылете из города)

Доступные опции пути:
* "Вылет" - вылететь из города/платформы/базы (в качестве параметра указывается список направлений)
* "Перелететь" - перелететь из города/платформы/базы (в качестве параметра указывается название места назначения)
* "В туннель" - перелететь с помощью рядом находящегося тоннеля (в качестве параметра указывается название места назначения)
* [x, y] - переместиться к координатам x, y, например [10, 10] или [31, 84], координаты от 0 до 100, согласно координатной сетке текущей карты игры (в качестве параметра указывается тип перемещения)
Типы перемещения: "Лететь к цели", "Быстро лететь к цели", "Переход между лабиринтами"

to_farm_path - Путь к месту фарма
to_base_path - Путь на базу
target_coords_range - Координаты перемещения игрока в заданной области фарма, если целей не найдено
repeat_cycle_forever - Повторять цикл фарма (вылет, убийство, возврат, сервис, разгрузка) бесконечно (true - чтобы включить, false - чтобы выключить)

Параметры точности поворота и назначения:
direction_bias - Точность поворота корабля
warp_bias - Какое перемещение по координатам считать перелётом в другую локацию (т.е. если координаты поменялись с 10 до 15 за раз - значит прыгнули на 5) (по координатной сетке)
target_bias - Отклонение от точки назначения, чем больше, тем раньше считается, что мы находимся в месте назначения (например, от точки обхода или города) (по координатной сетке)
stuck_difference - Минимальное расстояние, которое должен пройти корабль при перемещении за отведенное время (ниже), иначе считается, что корабль застрял (по координатной сетке)
stuck_delay - Задержка по времени в секундах, если позиция корабля за отведенное время не изменилась на определенное расстояние (выше), то считается, что корабль застрял

Параметры зоны:
stop_at_destination - Останавливаться при достижении точки перемещения (true - чтобы включить, false - чтобы выключить)
circle_area - Задание круговой области фарма или полигональной (true - круг, false - полигон)
Если задан полигон в качестве области фарма
area_points - Вершины (точки) для задания полигона области фарма (по координатной сетке)
Если задана окружность в качестве области фарма
area_coords - Центр области фарма
Следующие два параметра задают как бы бублик на карте для фарма
max_distance_away_from_area - Ограничение как далеко можно отходить от центра области фарма (по координатной сетке)
min_distance_from_area - Ограничение как близко можно подходить к центру области фарма (по координатной сетке)

max_farm_time - Максимальное время фарма в секундах
do_looting - Собирать лут (true - чтобы включить, false - чтобы выключить)
ignore_overweight - Игнорировать перегруз как сигнал к тому, что пора возвращаться обратно (true - чтобы включить, false - чтобы выключить)
do_drop_chests - Выкидывать сундуки перед полетом обратно (true - чтобы включить, false - чтобы выключить)

Параметры рыбалки:
fishing_spot_detection_precision - Точность определения рыболовного места (от 0 до 1, например 0.65)
fishing_spot_approach_distance - Дистанция приближения к рыболовному месту (от 0 до бесконечности)
fishing_spot_max_bad_fishing_tries - Количество попыток подлететь и начать рыбачить (если столкнули с места) за время одного вылета (от 0 до бесконечности)

Параметры убийства врагов:
enemy_types - Типы врагов для убийства ("neutral" - белые/нейтральные, "agressive" - красные/агрессивные, "special" - фиолетовые/реплоиды) (можно задавать через запятую ["neutral", "agressive"])
enemy_focusing_max_time - Максимальное время на то, чтобы взять цель в таргет в секундах (иначе переход к точке)
smart_targeting - Включить умный выбор целей (true - чтобы включить, false - чтобы выключить)
fire_when_smart_targeting - Включить стрельбу (спам залпа из всех орудий) при приблежении к цели при умном выборе целей (true - чтобы включить, false - чтобы выключить)
jump_forward_on_lose_target - Ускоряться вперед после убийства цели. Полезно, чтобы залутать без лишних кликов. (true - чтобы включить, false - чтобы выключить)
do_drop_chests - Сбрасывать сундуки по окончании фарма перед полетом обратно (true - чтобы включить, false - чтобы выключить)
stop_if_enemy_in_front_of_ship - Если движемся к следующей точке в области, то остановиться, если на пути есть враг (true - чтобы включить, false - чтобы выключить)
enemy_kill_timeout - Время на убийство одного врага в секундах (чтобы если сломается автоатака, или жук какой будет убегать постоянно - снять цель и попробовать снова)

death_check_delay - Время в секундах, как часто проверять жив игрок или же умер
after_death_wait_time_range - Диапазон времени в секундах, сколько ждать после смерти игрока перед началом нового цикла (например [300, 300] чтобы ждать 5 минут или [300, 3600] чтобы ждать от 5 до 60 минут)

abilities - Номера способностей, которые нужно прожать с началом фарма, например [1, 2, 3]

Горячие клавиши:
force_key - Форсаж вперед (расходуется охладитель)
force_forward_key - Ускорение вперед
force_left_key - Ускорение влево
force_right_key - Ускорение вправо
forward_key - Лететь вперед
break_target_key - Сбросить цель
fire_key - Залп из всех орудий
press_esc_after_radar_action - Нажимать клавишу Escape после каждого нажатия на радар, чтобы снимать потенциальные менюшки при кликах не туда (true - чтобы включить, false - чтобы выключить)
