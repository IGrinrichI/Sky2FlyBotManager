mode - Режим работы бота: "Рыбалка", "Убийство мобов в зоне", "Одуванчики", "Дерево"
next_preset - Название следующего пресета, который нужно запустить (если repeat_cycle_forever - true, то не запускает следующий, например "Туканы Лагуна")
check_state_in_city_before_farm - Выполнять проверку в городе в начале работы перед вылетом, как будто залетел в город (зарядка, скидывание на склад, и т.д., true - чтобы включить, false - чтобы выключить)

available_directions_to_undock - Направления, в которых вылетать из города "с" - север, "ю" - юг, "з" - запад, "в" - восток и их комбинации ("св", "сз", "юв", "юз") (выбирается случайно при вылете из города)

Доступные опции пути (алгоритма):
* "Вылет" - вылететь из города/платформы/базы (в качестве параметра указывается список направлений)
* "Перелететь" - перелететь из города/платформы/базы (в качестве параметра указывается название места назначения)
* "В туннель" или "В тоннель" - перелететь с помощью рядом находящегося тоннеля (в качестве параметра указывается название места назначения)
* "Пролететь через вихрь" - пролететь через вихрь в небе, находящийся в радиусе видимости радара (в качестве параметра указывается любое значение, например "")
* "Ожидание перелета" - ожидание изменения места нахождения, может быть полезно при заведении в сценарий в качестве ожидания его начала (в качестве параметра указывается любое значение, например "")
* "Пригласить в пати" - отправить приглашение присоединиться к группе (в качестве параметра указывается ник игрока, например "Чак_Норис")
* "Ожидание пати" - ожидание прихода приглашения присоединиться к группе (в качестве параметра указывается любое значение, например "")
* "Принять пати" - принять действующее приглашение присоединиться к группе (в качестве параметра указывается любое значение, например "")
* "Суицид" - прописать в чат команду /die для возврата в город и возвращается в город (в качестве параметра указывается любое значение, например "")
* "Начать диалог" - инициировать диалог, при нахождении рядом с NPC (в качестве параметра указывается любое значение, например "")
* "Выбрать опцию диалога" - выбор опции диалога в диалоговом окне с NPC (в качестве параметра указывается тип диалоговой опции и её порядковый номер среди такого же типа диалоговых опций, т.е. ["Выбрать опцию диалога", ["Стандарт", 2]] будет означать, что выберется вторая доступная опция среди опций типа "Стандарт", также можно указывать без порядкового номера, тогда будет браться первая попавшаяся, например ["Выбрать опцию диалога", "Закрыть"])
Типы диалоговых опций: "Стандарт", "Закрыть", "Квест", "Сдать", "Вопрос"
* [x, y] - переместиться к координатам x, y, например [10, 10] или [31, 84], координаты от 0 до 100, согласно координатной сетке текущей карты игры (в качестве параметра указывается тип перемещения)
Типы перемещения: "Лететь к цели", "Быстро лететь к цели", "Переход между лабиринтами"

to_farm_path - Путь (алгоритм) к месту фарма
to_base_path - Путь (алгоритм) на базу
target_coords_range - Координаты перемещения игрока в заданной области фарма, если целей не найдено
repeat_cycle_forever - Повторять цикл фарма (вылет, убийство, возврат, сервис, разгрузка) бесконечно (true - чтобы включить, false - чтобы выключить)

Параметры точности определения и приближения к тоннелям:
tunnel_detection_precision - Точность определения тоннеля (от 0 до 1, например 0.7)
tunnel_approach_distance - Дистанция приближения к тоннелю (от 0 до бесконечности, например 5)

Параметры точности определения и приближения к вихрям:
vortex_detection_precision - Точность определения вихря (от 0 до 1, например 0.2)
vortex_approach_distance - Дистанция приближения к вихрю (от 0 до бесконечности, например 10)

Параметры точности поворота и назначения:
direction_bias - Точность поворота корабля
warp_bias - Какое перемещение по координатам считать перелётом в другую локацию (т.е. если координаты поменялись с 10 до 15 за раз - значит прыгнули на 5) (по координатной сетке)
target_bias - Отклонение от точки назначения, чем больше, тем раньше считается, что мы находимся в месте назначения (например, от точки обхода или города) (по координатной сетке)
stuck_difference - Минимальное расстояние, которое должен пройти корабль при перемещении за отведенное время (ниже), иначе считается, что корабль застрял (по координатной сетке)
stuck_delay - Задержка по времени в секундах, если позиция корабля за отведенное время не изменилась на определенное расстояние (выше), то считается, что корабль застрял
scale_in_out_radar_delay - Задержка по времени в секундах после приближения/отдаления радара (от 0 до бесконечности, например 1) (иногда замечается, что необходимо немного подождать прежде чем после масштабирования радара что-то делать)

Параметры зоны:
stop_at_destination - Останавливаться при достижении точки перемещения (true - чтобы включить, false - чтобы выключить)
circle_area - Задание круговой области фарма или полигональной (true - круг, false - полигон)
Если задан полигон в качестве области фарма
area_points - Вершины (точки) для задания полигона области фарма (по координатной сетке)
Если задана окружность в качестве области фарма
area_coords - Центр области фарма
Следующие два параметра задают как бы бублик на карте для фарма
max_distance_away_from_area - Ограничение как далеко можно отходить от центра области фарма (по координатной сетке)
min_distance_from_area - Ограничение как близко можно подходить к центру области фарма (по координатной сетке)
positioning_to_the_center - Позиционировать корабль в центре области (true - чтобы включить, false - чтобы выключить) (тут имеется ввиду рыбное место, дерево)

max_farm_time - Максимальное время фарма в секундах
delay_between_farm_attempts - Задержка по времени между итерациями фарма в секундах (чем выше, тем меньше нагрузка на систему, но и медленнее фарм, для убийства мобов - между атаками и проверкой , что атакует, для рыбалки - что рыбачит, для дерева - что рубит и лутанием, одуванчики - задержка перед перелетом к следующему одуванчику)
action_timeout - Максимальное время ожидания различных действий, например открытия окна склада/сервиса/трюма (от 0 до бесконечности, например 5)
do_looting - Собирать лут (true - чтобы включить, false - чтобы выключить)
loot_distance - Дальность сбора лута (от 0 до примерно 100, дальше смысла нет, например 30)
ignore_overweight - Игнорировать перегруз как сигнал к тому, что пора возвращаться обратно (true - чтобы включить, false - чтобы выключить)
do_drop_chests - Выкидывать сундуки перед полетом обратно (true - чтобы включить, false - чтобы выключить)

Параметры рыбалки:
fishing_spot_detection_precision - Точность определения рыболовного места (от 0 до 1, например 0.65)
fishing_spot_approach_distance - Дистанция приближения к рыболовному месту (от 0 до бесконечности)
fishing_spot_max_bad_fishing_tries - Количество попыток подлететь и начать рыбачить (если столкнули с места) за время одного вылета (от 0 до бесконечности)

Параметры сбора одуванчиков:
dandelion_detection_precision - Точность определения одуванчиков (от 0 до 1, например 0.85)
dandelion_approach_distance - Дистанция приближения к одуванчику (от 0 до бесконечности)

Параметры убийства врагов:
kill_enemies - Убивать ли врагов (true - чтобы включить, false - чтобы выключить)
spam_attack - Спамить атаку (true - чтобы включить, false - чтобы выключить) (работает пока только для дерева, не смотрит в kill_enemies)
spam_attack_interval - Интервал атак при спаме (от 0 до бесконечности, например 0.17)
enemy_types - Типы врагов для убийства ("neutral" - белые/нейтральные, "agressive" - красные/агрессивные, "special" - фиолетовые/реплоиды, "boss" - боссы) (можно задавать через запятую ["neutral", "agressive"])
enemy_focusing_max_time - Максимальное время на то, чтобы взять цель в таргет в секундах (иначе переход к точке)
smart_targeting - Включить умный выбор целей (true - чтобы включить, false - чтобы выключить)
fire_when_smart_targeting - Включить стрельбу (спам залпа из всех орудий) при приблежении к цели при умном выборе целей (true - чтобы включить, false - чтобы выключить)
jump_forward_on_lose_target - Ускоряться вперед после убийства цели. Полезно, чтобы залутать без лишних кликов. (true - чтобы включить, false - чтобы выключить)
do_drop_chests - Сбрасывать сундуки по окончании фарма перед полетом обратно (true - чтобы включить, false - чтобы выключить)
stop_if_enemy_in_front_of_ship - Если движемся к следующей точке в области, то остановиться, если на пути есть враг (true - чтобы включить, false - чтобы выключить)
enemy_kill_timeout - Время на убийство одного врага в секундах (чтобы если сломается автоатака, или жук какой будет убегать постоянно - снять цель и попробовать снова)

death_check_delay - Время в секундах, как часто проверять жив игрок или же умер
after_death_wait_time_range - Диапазон времени в секундах, сколько ждать после смерти игрока перед началом нового цикла (например [300, 300] чтобы ждать 5 минут или [300, 3600] чтобы ждать от 5 до 60 минут)

abilities - Номера способностей, которые нужно прожать с началом фарма, например [1, 2, 3]

Горячие клавиши:

Доступные специальные клавиши: Ctrl, LCtrl, RCtrl, Shift, LShift, RShift, Alt, LAlt, RAlt, Esc, Tab, PageUp, PageDown, Back, Clear, CapsLock, End, Home, Insert, Apps
press_esc_after_radar_action - Нажимать клавишу Escape после каждого нажатия на радар, чтобы снимать потенциальные менюшки при кликах не туда (true - чтобы включить, false - чтобы выключить)

force_key - Форсаж вперед (расходуется охладитель), по умолчанию "f"
fire_key - Залп из всех орудий, по умолчанию " "
auto_use_all_key - Вкл. автоиспользование всех техустройств, по умолчанию "~"
break_target_key - Сбросить цель, по умолчанию "z"
activate_ability_1_key - Активное умение 1, по умолчанию "[Ctrl]1"
activate_ability_2_key - Активное умение 2, по умолчанию "[Ctrl]2"
activate_ability_3_key - Активное умение 3, по умолчанию "[Ctrl]3"
activate_ability_4_key - Активное умение 4, по умолчанию "[Ctrl]4"
activate_ability_5_key - Активное умение 5, по умолчанию "[Ctrl]5"
activate_ability_6_key - Активное умение 6, по умолчанию "[Ctrl]6"
activate_ability_7_key - Активное умение 7, по умолчанию "[Ctrl]7"
activate_ability_8_key - Активное умение 8, по умолчанию "[Ctrl]8"
activate_ability_9_key - Активное умение 9, по умолчанию "[Ctrl]9"
force_left_key - Ускорение влево, по умолчанию "x"
force_right_key - Ускорение вправо, по умолчанию "c"
force_forward_key - Ускорение вперед, по умолчанию "q"
radar_out_key - Масштаб радара мельче, по умолчанию "[Ctrl]-"
radar_in_key - Масштаб радара крупнее, по умолчанию "[Ctrl]+"
inventory_key - Трюм/корабль, по умолчанию "i"
